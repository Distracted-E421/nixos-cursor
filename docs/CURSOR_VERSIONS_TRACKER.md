## Custom Modes Storage

Custom modes in 2.0.x are stored in:
- **Windows**: `%APPDATA%\Cursor\User\globalStorage\state.vscdb`
- **macOS**: `~/Library/Application Support/Cursor/User/globalStorage/state.vscdb`
- **Linux**: `~/.config/Cursor/User/globalStorage/state.vscdb`

## Our Strategy

Since Cursor removed custom modes in 2.1+, we implement **external custom modes** via Cursor Studio:

1. **Mode definitions** stored in `~/.config/cursor-studio/modes/`
2. **Injection** via `.cursorrules` and `.cursor/rules/*.mdc`
3. **AI workspace** context via `.ai-workspace/hints.md`

This allows using **any version** with full mode functionality.
